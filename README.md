# franksfc.github.io

Personal CV homepage for Feichen Song, built as a lightweight static site.

## Publish a Blog Post

1. Add a Markdown file under `posts/`, for example `posts/2026-05-10-my-note.md`.
2. Add a matching entry to `posts/manifest.json`.
3. Push the repository to GitHub Pages.

No Hexo or build step is required.
